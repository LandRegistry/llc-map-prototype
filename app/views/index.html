Search for local land charges

Search is a public facing service that allows users to search an area for local land charges.

This prototype shows an iterated design for an interactive map using the existing openlayers js library, re-styled to use GDS components. It includes a pattern overlay and an example using keyboard control to select an existing property boundary

Anyone can use the service but an account is currently mandatory. Free and paid searches are both available; free searches list charges found but do not show the area the charge extent covers, while paid searches generate a PDF (JSON, CSV or XML are also available) with all charge and extent information which is then saved for 6 months and can be repeated in that time for free.
