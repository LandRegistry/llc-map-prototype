Search for local land charges

Search for local land charges is a public facing service that allows users to search an area for local land charges.

This prototype shows design iterations of the existing interactive map that is used to define a custom search area. The prototype was built using the existing openlayers js library, re-styled to use GDS components. It includes a pattern overlay and an example using keyboard control to select an existing property boundary.

Anyone can use the live service but an account is currently mandatory. Free and paid searches are both available; free searches list charges found but do not show the area the charge extent covers, while paid searches generate a PDF (JSON, CSV or XML are also available) with all charge and extent information which is then saved for 6 months and can be repeated in that time for free.

Proof of concept

This is a simple OpenLayers map using Local Area District (LAD) boundaries to test navigation and area selection using the keyboard. External buttons are used to navigate the map and to select the area at the center of the map. The coordinates of this central point are then used to find the LAD (using getFeaturesAtCoordinate) which is then styled to show it has been selected. Sample js code.
